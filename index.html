Raytracer

C++

SDL

This raytracer was a nice project to work on. A raytracer is able to visualize objects onto the screen by calculating the light value of each ray that is cast for each pixel. As this is now a hot topic in the gaming industry, it was a good learning foundation. I learned several techniques about shading and how rays intersect with certain shapes.

Highlights

  • Perspective
  • Lightning
  • physically based rendering
